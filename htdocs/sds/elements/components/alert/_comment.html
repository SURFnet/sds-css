Note that .page-container is applied twice on this demo page, which is not desirable, but just convenient. For the alert component we normally assume to have full page-width so .page-container, is needed inside. For the demo page it's convenient to have .page-container on a high level, so for this demo page we make breakout the alert.

Note that the alert doesn't always need a .page-container inside. If it's placed inside an element that doesn't have full page-width, for example inside the alert component.

Note that applying icon-color is done by applying .bg to an element inside the svg. The lines are always white (no extra class needed).
